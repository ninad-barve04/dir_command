# dir_command

This project is a modified implementation of linux dir command in C.

Installation:

'''sh
 // Clone this repository in a folder and navigate to it
 make
 chmod -rwx ./dirx
 sudo mv ./dirx /bin/

'''

The options are:

* Sort

-s name

-s size

-s date

* Revese sort order

-r

* Recursive

-R

* Display as tree

-t

* Show only files of particular type (txt/c/pdf)

-f [filetype]
